# Multi-Agent AI-Powered Mock Interviewer using CrewAI

An intelligent mock interview system powered by CrewAI that simulates realistic job interviews with AI-driven feedback and evaluation.

## Features
- Realistic interview simulation
- Real-time feedback
- Performance analytics
- Multiple interview roles
- Personalized improvement suggestions

## Tech Stack
- CrewAI
- Python
- Natural Language Processing
- Speech Recognition

## Installation
```bash
pip install -r requirements.txt
```

## Usage
```bash
python main.py
```
